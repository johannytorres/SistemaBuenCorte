# Capa de Lógica de Negocio (BLL — Business Logic Layer)

Reglas y procesos del negocio. Orquesta las operaciones entre la
presentación y el acceso a datos.

**Aquí va:**
- Services (ej. `AuthService`, `ProductoService`)
- Validaciones de negocio
- DTOs y lógica de transformación

**Depende de:** `SistemaBuenCorte.DAL`
**Usada por:** `SistemaBuenCorte.Web`
